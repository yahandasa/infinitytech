## 1) Performance Optimization (CRITICAL)

The website currently feels laggy. Improve performance with the following strict rules:

* Reduce heavy visual effects:

  * Limit backdrop-blur to a maximum of blur(8px)
  * Remove excessive glow and large shadows
  * Avoid layered filters and complex CSS effects

* Optimize animations:

  * Do NOT use "transition: all"
  * Use only:

    * transform
    * opacity
  * Duration: 0.2s–0.3s max

* Use GPU acceleration where needed:

  * Apply transform: translateZ(0) or will-change: transform on animated elements

* Disable or remove:

  * Particle backgrounds
  * Heavy canvas animations
  * Continuous infinite animations

* Optimize media:

  * Enable lazy loading for images
  * Compress images
  * Avoid autoplay videos in hero section
